# gh meow

Get a cat to meow at you.

## Install

```
gh extension install BRAVO68WEB/gh-meow
```

## Usage

```bash
$ gh meow
 /\_/\
( o.o )
 > ^ <
> Meow!

$ gh meow --help

gh meow is a command to print cats in the terminal
                You can print different types of cats by passing the following arguments:
                - xl: Chonk XL Meow!
                - nyan: Nyan Cat Meow!
                - hey: Hey Cutie!
                - cute: Cute Meow!
                - sleep: Sleeping Meow!
                - no argument: Meow!

Usage:
  gh meow [flags]

Flags:
  -h, --help   help for gh
```
